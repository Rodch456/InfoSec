1. Resident (Normal User)

2. Barangay Official
Primary capabilities:
View all resident incident reports
Update the status of reports (e.g., Pending ‚Üí In Progress ‚Üí Resolved)
Request upload of memos or ordinances (subject to Admin approval)
View memos, ordinances and report summaries (but not generate exports)
View their own activity logs (optional feature)
Restrictions:
Cannot create user accounts
Cannot approve memo/ordinance requests
Cannot generate logs, exports, or advanced reports
3. Admin
Primary capabilities:
Full access to the system
Account management:
Create Barangay Official accounts
Reset passwords
Change role status (active/inactive)
Report management:
View all reports
Update report status
Ask for more information (follow-up questions)
Manage memos & ordinances:
Upload memos & ordinances directly
Review & approve/reject submission requests from officials
View and manage logs & generate reports:
Complete system logs (security + actions)
Filter logs by user, date, action type
Export logs (CSV/PDF)
Generate analytics & progress reports:
Summaries (counts per status/category/priority)
Date filters
Export as CSV/PDF
Restrictions:
None (superuser)
